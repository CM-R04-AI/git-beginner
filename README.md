# git-beginner

## steps
### initialize
git init

### add local Repo
#### add all files
git add .  
#### add single file
git add \<file name\> 
#### add to a local Repo
git commit -m "\<your msg\>"

### check remote Repo
git remote -v

### add remote Repo
'''
git remote add local <remote url>
git branch -M <brach name ex. main>
git push -u local <brach name ex. main>
'''
